# 项目

激光雷达运行数据管理系统

  • 技术栈
    SpringBoot SSM MySQL Redis Nginx Redis RabbitMQ

  • 项目描述
    基于SpringBoot框架和SSM架构的激光雷达运行数据管理系统，支持通过网⻚、微信小程序访问激光雷达台站的实时运行数据，提供用戶管理、设备数据管理、远程控制设备服务。

  • 工作内容：

    • 所有信息数据由台站本地的上位机系统实时更新到云数据库，本项目服务端使用Mybatis框架访问数据库，访问最新数据渲 染到前端。
    • 对于访问量较大的部分数据，使用Redis作为缓存服务器，请求首先访问缓存，不成功则访问数据库，提高了访问效率，也 能保证数据库和缓存的数据一致性。
    • 使用RabbitMQ中间件作为远程控制设备的服务端。激光雷达台站本地上位机系统作为消息消费端，B端作为消息生产端， 实现通过互联网远程控制设备。
    • 该项目目前已正式部署运行，为国家基金项目“子午工程”提供服务。
